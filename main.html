An example of an external css stylesheet

what you'll learn.

  • how to make a .css style sheet
  • how to include it in your .html file

how to make a .css style stylesheet

open your preffered text editor, notepad etc. make a file and call it stylesheet.css css works by taking an input, let's say p. And then changing different things about it, how big it is, what colour and so on.

to do this you'll format the code like this,

      p { color = red; }
      h1 { color = green; }

the syntax

      (your html attribute like p) { (your change to it); }

how to include it in your html file

      link rel="stylesheet" href="stylesheet.css"
